# X32 Routing Pro PWA

This is an installable Progressive Web App for iPad, Mac, Windows, and Android.

## How to use locally
Open `index.html` in a browser to test the app.

## Best way to install on iPad
A PWA needs to be hosted on a web server with HTTPS for full offline install behavior.
Upload this folder to Netlify, Vercel, GitHub Pages, or your website.
Then on iPad:
1. Open the web address in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

## Features
- 32 editable channels
- Channel item field
- XLR/Line input type
- Phantom power checkbox
- L/C/R pan selector
- Row color picker
- Notes field
- Multiple saved worksheets in local browser storage
- Duplicate worksheets
- Export/import worksheet JSON
- Print / Save to PDF one-pager
- Offline-capable when hosted and installed
